# 🎙️ Voice Activated Personal Assistant

## 📌 Project Description

This project was developed as part of the **QSkill Python Development Internship**.

It is a desktop-based Personal Assistant built using Python. The assistant can perform common tasks such as speaking responses, opening applications, opening websites, managing reminders, and performing calculations.

---

## 🚀 Features

- 🗣️ Text-to-Speech
- 👋 Greeting
- ⏰ Current Time
- 📅 Current Date
- 🌐 Open Google
- ▶️ Open YouTube
- 💻 Open GitHub
- 📂 Open Notepad
- 🖩 Open Calculator
- ⚫ Open Command Prompt
- 🎨 Open Paint
- 📁 Open File Explorer
- 📝 Add Reminder
- 📋 Show Reminders
- 🧮 Built-in Calculator
- 📖 Help Menu
- 👋 Exit Assistant

---

## 🛠 Technologies Used

- Python
- pyttsx3
- webbrowser
- subprocess
- datetime
- os
