# Voice Assistant - Observations

## Objective

Developed a desktop Voice Assistant using Python.

## Features Implemented

- Text-to-Speech
- Website Automation
- Desktop Application Launcher
- Reminder System
- Calculator
- Help Menu

## Result

The assistant successfully performs desktop automation, file handling, and user interaction through typed commands and spoken responses.

## Conclusion

This project demonstrates Python automation, text-to-speech integration, and modular programming concepts.
